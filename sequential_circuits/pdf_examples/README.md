# Sequential Verilog Examples from Uploaded PDF

Cleaned/adapted educational RTL corresponding to the sequential-design sections of the 98-page **Verilog Examples** PDF.

Covered topics:

- D Flip-Flop with asynchronous reset
- D Flip-Flop with synchronous reset
- SR, JK and T Flip-Flops
- Universal Shift Register
- 4-bit LFSR
- 4-bit asynchronous/ripple counter topic
- 1010 sequence detectors: Mealy/Moore, overlapping/non-overlapping
- Booth multiplier topic
- Synchronous FIFO
- Asynchronous FIFO with Gray-coded pointer synchronization

The PDF source contains multiple implementation variants for FIFO designs and some mixed Verilog/SystemVerilog constructs. Repository files normalize syntax and module naming where needed; they are cleaned/adapted implementations corresponding to the source topics, not guaranteed byte-for-byte reproductions of OCR text.
